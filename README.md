# Customer Support Analytics Dashboard

## Overview
This project analyzes customer support data using a simulated dataset to evaluate service performance and customer satisfaction. The dashboard provides insights into ticket trends, issue categories, resolution times, and escalation patterns.

> **Note:** This project uses a simulated dataset created for learning and portfolio purposes. It does not represent data from any real organization.

## Objectives
- Analyze customer support tickets
- Identify common customer issues
- Monitor resolution performance
- Measure customer satisfaction
- Evaluate escalation trends
- Recommend service improvements

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards

## Key Metrics
- Total Support Tickets
- Resolution Time
- Customer Satisfaction Score (CSAT)
- Escalation Rate
- Issue Category Distribution
- Ticket Status
